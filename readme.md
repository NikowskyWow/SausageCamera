A lightweight utility for unlocking the maximum camera zoom distance on Warmane.

---

## 🚀 Features
* **Camera Unlock:** Bypasses default UI limits and forces the `cameraDistanceMaxFactor` to its maximum for the 3.3.5a client.


---

## 🛠 Installation

1. Download the latest version of the addon.
2. Extract the folder into your World of Warcraft directory:
   `World of Warcraft/Interface/AddOns/`
3. Ensure the folder name is exactly **SausageCameraFix**.
4. Restart the game or type `/reload` in-game.

---

## 🎮 How to Use

List the slash commands or UI elements here:

* `/sausagecamera` - Opens the main configuration window.
* **UI:** Click the "Unlock Camera" button inside the menu to instantly unlock the zoom limits. Then use your mouse wheel to zoom out.
